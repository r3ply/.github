# r3ply

An open source commenting system for websites that allows them to receive comments via email, while preserving privacy.

If you're curious how it works or to learn more about the features, please visit [r3ply.com](https://r3ply.com) to learn more. 

Source code:

- [r3ply](https://github.com/r3ply/r3ply): the r3ply mono repo, containing the r3ply library, schemas, wasm helpers, and applications
- [r3ply zola theme](https://github.com/r3ply/r3ply-zola-theme): a zola theme for r3py that makes adding commenting to your site easy
